# Part 3. Claude Code 사용 시 작업 시간 추정

> Claude Code 기준 각 작업의 예상 소요 시간입니다.  
> 기준: 코드 생성 + 코드 리뷰 수정 + ESLint 통과 + 로컬 동작 확인까지 포함  
> **사람이 검토하는 시간은 별도로 추가해야 합니다.**

---

## 시간 등급 기준

| 등급 | 시간 | 작업 규모 |
|------|------|-----------|
| XS | ~15분 | 유틸 함수 1개, 상수 파일 |
| S | ~30분 | 단순 컴포넌트 1개, 훅 1개 |
| M | 1~2시간 | 복합 컴포넌트, 페이지 1개 |
| L | 2~4시간 | 기능 단위 (CRUD 1개), 서비스 + 타입 + UI |
| XL | 4~8시간 | 전체 모듈 (인증 시스템, DB 스키마 설계) |
| XXL | 1~2일 | 대형 기능 (결제, 전체 API 연동) |
