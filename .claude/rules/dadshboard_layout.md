## 전체 페이지 구조

페이지는 좌측 고정 사이드바 + 우측 메인 영역의 2-column 레이아웃으로 구성한다.
사이드바는 화면 전체 높이를 차지하며 스크롤되지 않는다.
메인 영역만 수직 스크롤된다.

### 사이드바 내부 구조 (위→아래 순서)
① 브랜드 로고 영역 (상단 고정)
② 로그인 사용자 프로필 카드 (아바타 + 이름 + 역할 텍스트)
③ 내비게이션 메뉴 리스트 (아이콘 + 라벨, 활성 항목 강조)
④ 하단 고정 영역:
   - 새 보고서 생성 버튼 (full-width, 강조색)
   - 매뉴얼 링크
   - 로그아웃 링크

### 메인 영역 상단 헤더 (sticky)
[ 검색바(중앙 확장) ]  [ 알림 아이콘 ] [ 설정 아이콘 ] [ 사용자 역할 텍스트 ]
→ 좌우 양끝 정렬. 검색바가 남은 공간을 채운다.

---

## 섹션 1 — KPI 카드 행

3개의 카드가 동일한 너비로 가로 배열 (1:1:1 비율).
각 카드 내부 구조:
  - 상단: 아이콘(좌) + 변동률 뱃지(우)
  - 중단: 주요 수치 (대형 폰트)
  - 하단: 목표 대비 프로그레스바 + 달성률 텍스트
세 번째 카드는 경고 상태로, 테두리 색상이 다르고
우상단에 CRITICAL 상태 뱃지가 붙는다.

---

## 섹션 2 — 차트 행

좌(넓음) + 우(좁음) 2열 구성. 비율은 약 2:1.

[좌] Combo Chart 카드:
  - 카드 헤더: 제목(좌) + 연도 토글 버튼(우)
  - 범례: 두 항목 (막대 색상 샘플 + 라벨)
  - 차트 본체: 막대(월간 매출) + 꺾은선(누적 목표), Chart.js 사용
  - 특정 막대 위에 플로팅 라벨(DROP / GROWTH) 표시

[우] 도넛 차트 카드:
  - 카드 헤더: 제목
  - 도넛 차트 중앙에 "TOTAL CATEGORIES N" 텍스트 삽입
  - 하단 범례: 색상 점 + 카테고리명 + 비율(%) + 변동률(±%)
    → 항목마다 1줄씩, 세로 리스트

---

## 섹션 3 — 재고 부족 현황 패널

카드 헤더 행:
  [ 제목 텍스트 ] [ ACTION REQUIRED 뱃지 ]  →  [ 전체 관리 링크(우측 정렬) ]

본문: 3개의 재고 아이템 카드가 가로 균등 배열.
각 아이템 카드 내부:
  - 상단: 썸네일 이미지(좌) + 제품명 + 카테고리(우)
  - 중단: RISK LEVEL 라벨(좌) + 상태 텍스트(우, 색상 강조)
  - 하단: 리스크 게이지 프로그레스바 (채워진 비율로 위험도 표현)
  - 최하단: "잔여: N" (좌) + "임계: N" (우) 텍스트

---

## 섹션 4 — 최근 주문 내역 테이블

카드 헤더 행:
  [ 제목 + 실시간 스트림 서브텍스트 ]  →  [ 검색바 ] [ 필터 버튼 ] [ CSV 내보내기 버튼 ]

테이블 컬럼 순서 (좌→우):
  주문번호 | 고객정보 | 상품상세 | 결제금액 | 결제방식 | 배송상태 | 액션(…)

고객정보 셀: 아바타 원형 이미지 + 이름 + 회원등급 뱃지 (2줄 구성)
결제방식 셀: 결제사 로고/텍스트 뱃지
배송상태 셀: 색상 점 + 상태 텍스트 인라인 뱃지
액션 셀: 우측 정렬, 점 3개(…) 아이콘 버튼

테이블 푸터:
  [ SHOWING N OF N,NNN RECORDS (좌) ]  →  [ PREVIOUS ] [ NEXT PAGE ] (우)

## 반응형 레이아웃 규칙

### PC (≥1280px) — 기본 레이아웃
- 사이드바: 고정 표시 (190px 너비)
- KPI 카드: 3열
- 차트: 2열 (2:1 비율)
- 재고 카드: 3열
- 테이블: 모든 컬럼 표시

### 태블릿 (768px ~ 1279px)
- 사이드바: 아이콘만 표시 (60px), hover 시 라벨 툴팁
- KPI 카드: 2열 + 1개 아래 행
- 차트: 1열 (세로 스택)
- 재고 카드: 2열 + 1개 아래 행
- 테이블: 보조 컬럼(결제방식) 숨김

### 모바일 (≤767px)
- 사이드바: 완전히 숨김 → 상단에 햄버거 버튼 추가
  클릭 시 전체화면 오버레이 메뉴로 슬라이드인
- KPI 카드: 1열 전체 너비
- 차트: 1열 전체 너비
- 재고 카드: 1열 전체 너비
- 테이블 → 카드형으로 변환:
  각 행이 독립 카드로 렌더링됨
  주문번호 + 상태 뱃지를 카드 헤더에, 나머지 정보를 하단에 2열 그리드로 배치

## 인터랙션 명세

- 모든 카드: hover 시 위로 살짝 부상 (translateY -2px)
- 사이드바 메뉴 활성 항목: 좌측 세로 강조선 + 배경 하이라이트
- CRITICAL 뱃지: pulse 애니메이션 (크기/투명도 반복)
- 프로그레스바: 페이지 로드 시 0 → 목표값으로 채워지는 CSS 트랜지션
- 테이블 행: hover 시 행 배경 미세 강조
- 모바일 햄버거 메뉴: 열림/닫힘 슬라이드 트랜지션
